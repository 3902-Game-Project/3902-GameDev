Program controls:
  Arrow keys and WASD keys move the player.
  Z and N keys cause the player to attack.
  T (previous) and Y (next) keys cycle between what block is drawn.
  Q to quit the game.
  R to reset to main menu.
  Enter to go from main menu to game.

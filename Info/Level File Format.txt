Level files are *.csv files.
The file describes a grid of blocks / entities.
Each line in the file is a row of blocks in the game.
Each cell (specific row and column) in the game is separated by "," in the file.
If there are multiple blocks in the same cell (for example, a background block and an enemy), they are separated by ";".

0 = empty
3 = player

non colliding blocks:
5 = sand
6 = red sand
7 = wood plank

colliding blocks:
1:X = log
  X = variation:
    0 = wall
    1 = corner
8:X = rock
  X = variation:
    0 = wall
    1 = corner
2:X = door to other room
  X = pairing id number; door connects to other door with same pairing ID
